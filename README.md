# API-CAR

Este proyecto tiene como objetivo crear una API en Python que permita almacenar información de carros en un archivo de texto con formato JSON. La API permite realizar operaciones POST para guardar nuevos carros y GET para obtener la lista de carros almacenados.

## Requisitos y programas vistos en el curso Porgramación Backend

- Python actualizado
- Flask
- Postman (opcional, para realizar pruebas)
